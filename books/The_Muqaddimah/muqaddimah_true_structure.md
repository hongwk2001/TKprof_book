**Note on the Source Text (`The_Muqaddimah_Original.txt`):**
The original Arabic text file was compiled by downloading and merging five separate OCR'd text files (`1_djvu.txt` through `5_djvu.txt`) from the Internet Archive (`https://archive.org/download/1_20230830_20230830_1222/`). Because it is an automated OCR dump of multiple physical volumes merged together, the file contains formatting inconsistencies, repeated Table of Contents sections, and out-of-order text blocks, which makes programmatic parsing of the structure challenging.
